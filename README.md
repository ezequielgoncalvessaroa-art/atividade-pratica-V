# ATIVIDADE PRÁTICA V - Desenvolvimento Web I

Projeto desenvolvido para a disciplina de Desenvolvimento Web I.

## Descrição

Este projeto consiste na criação de uma interface simples de rede social utilizando HTML e Pico CSS, contendo:

* Página de Login (`index.html`)
* Página de Cadastro (`cadastro.html`)
* Página de Feed (`feed.html`)

Todas as páginas foram desenvolvidas de forma responsiva para funcionamento em diferentes dispositivos.

## Tecnologias Utilizadas

* HTML5
* Pico CSS

## Funcionalidades

### Login

* Campo de e-mail
* Campo de senha
* Link para cadastro
* Link para recuperação de senha

### Cadastro

* Nome
* E-mail
* Usuário
* Senha
* Confirmação de senha
* Gênero
* Data de nascimento

### Feed

* Cabeçalho de navegação
* Área para criação de post
* Exibição de posts
* Curtidas e comentários
* Opção de sair
